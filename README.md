# smosummary
a lightweight C++ sum of best (SoB) calculator and stopwatch for super mario odyssey speedrunners.

## 🚀 Features
- **accurate timing:** precision stopwatch logic.
- **smart flags:** handles `-s` (start), `-h` (help), and even catches your typos.
- **arch native:** available on the AUR.
